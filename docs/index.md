# Claude Code Plugins for Nixtla

**Built by Intent Solutions io**

> **Status**: Early Concept Only – Initial plugin ideas for potential collaboration with Nixtla. Not endorsed by or affiliated with Nixtla. Subject to review and iteration.

## Empowering time series forecasting through AI-assisted development

Intent Solutions io partners with Nixtla to explore how Claude Code's natural language capabilities can accelerate adoption and implementation of the Nixtla ecosystem – from TimeGPT's foundation models to the comprehensive statistical and neural forecasting libraries.

## Why This Exists

The Nixtla ecosystem offers state-of-the-art time series forecasting across multiple paradigms. Claude Code plugins reduce the friction between "knowing about these tools" and "having them running in production." We're exploring how natural language commands can generate boilerplate, scaffold benchmarks, and template services – letting data scientists focus on their domain expertise rather than implementation details.

## Initial Plugin Concepts

We've drafted three plugin concepts to explore collaboration opportunities. Each represents a different workflow pain point that Claude Code could address.

### [TimeGPT Quickstart Pipeline Builder](plugins.md#1-timegpt-quickstart-pipeline-builder)
Transform natural language requirements into a working TimeGPT forecasting pipeline in minutes. Perfect for rapid prototyping and customer onboarding scenarios where speed-to-first-forecast matters.

### [Nixtla Bench Harness Generator](plugins.md#2-nixtla-bench-harness-generator)
Generate comprehensive benchmark harnesses that compare StatsForecast, MLForecast, NeuralForecast, and TimeGPT on your specific dataset. Make informed model selection decisions with reproducible, documented comparisons.

### [Forecast Service Template Builder](plugins.md#3-forecast-service-template-builder-fastapi--nixtla)
Scaffold production-ready FastAPI services that expose Nixtla models through REST endpoints. Includes containerization, request validation, and deployment configurations out of the box.
